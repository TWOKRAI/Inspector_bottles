# Анализ выбора Depth-камеры — Уровень 2/3 (3D-зрение, SLAM, obstacle avoidance)

> **Ключевое требование:** точная depth-карта для obstacle avoidance + SLAM на колёсном indoor-роботе.
> **Контекст:** Jetson Orin NX (100 TOPS) + Orin Nano (40 TOPS) = 166 TOPS на борту. Onboard AI на камере **не нужен**.
> **Рынок:** Россия, май 2026. Санкционные ограничения учтены.

---

## Технологии depth-камер — краткий ликбез

### Active IR Stereo (основная технология 2026)

```
[IR проектор] ─── ИК-точечный паттерн ──→ сцена
                                            ↑
[Левая IR камера] ←── отражённый паттерн ──┘
[Правая IR камера] ←── отражённый паттерн ──┘
        │                    │
        └──── диспаратность ─┘ → depth-карта
```

- Проецирует ИК-паттерн (точки/полосы) на сцену
- Две IR-камеры видят смещение паттерна → вычисляют глубину по диспаратности
- **Плюсы:** работает в темноте (ИК), хорошая точность на 0.1-10 м
- **Минусы:** на солнце ИК-паттерн забивается, на гладких поверхностях (стекло, зеркала) — артефакты
- **Используют:** Orbbec Gemini 335/335L, Intel RealSense D435i/D455, OAK-D Pro

### Passive Stereo

- Как человеческое зрение — две камеры без проектора
- **Плюсы:** дёшево, работает на солнце
- **Минусы:** не работает в темноте, плохо на однотонных поверхностях
- **Используют:** OAK-D Lite, OAK-D S2

### Time-of-Flight (ToF)

- Камера излучает импульс света и замеряет время возврата
- **Плюсы:** быстро, равномерное depth по всему кадру
- **Минусы:** короткая дальность (до 5 м), multipath interference, дорого
- **Используют:** OAK-D SR, Orbbec Femto Bolt/Mega, Azure Kinect (EOL)

**Для indoor-робота оптимально: Active IR Stereo** — баланс цены, точности и диапазона.

## Сравнение кандидатов

### 1. Orbbec Gemini 335

| Параметр | Значение |
|----------|---------|
| **Технология** | Active IR Stereo (850nm проектор) |
| **Depth разрешение** | до 1280×800 @ 30fps |

| **Диапазон** | 0.10 – 20 м (оптимально 0.26 – 3 м) |
| **Точность RMSE @ 2м** | < 1.5% |
| **Baseline** | 50 мм |
| **FOV (H×V)** | 90° × 65° |
| **IMU** | 6 DoF (BMI270) |
| **Onboard** | MX6800 ASIC (depth на камере) |
| **Питание** | < 3 Вт (USB 3.0) |
| **Размеры** | 90 × 25 × 30 мм |
| **Вес** | 97 г |
| **IP-защита** | IP5X (пыль) |
| **USB** | USB 3.0 Type-C |
| **Цена MSRP** | ~$264 |

**Плюсы:**
- Компактный, лёгкий, низкое энергопотребление
- MX6800 разгружает Jetson — depth вычисляется аппаратно на камере
- Широкий FOV 90°×65° — лучше всех конкурентов
- Минимальная дистанция 10 см — видит прямо под роботом
- Active + Passive stereo: работает и в темноте, и при ярком свете

**Минусы:**
- 50mm baseline — на 3+ метрах точность падает быстрее чем у 335L
- RGB ограничен 1280×800 (достаточно для робота, но не для фото)
- Нет IP65 (только пыль, без воды)

**Плюсы:**
- 4K RGB — лучшее качество цветного изображения
- Myriad X VPU — можно запускать модели прямо на камере (MobileNet, YOLO)
- IP66 — лучшая защита
- 9 DoF IMU (если повезёт с партией)
- Аппаратный H.264/H.265 encoder

**Минусы:**
- **7.5-15 Вт** — в 3-5 раз больше Orbbec. Критично для робота на аккумуляторе

- **Переплата за ненужные фичи** — 4K RGB, VPU, encoder
- **Pointcloud rate drop** — известный баг: после ~1 часа частота pointcloud падает с 30 Hz до 1 Hz
- **IMU лотерея** — может попасться 6-axis BMI270 вместо 9-axis BNO085
- **Своя экосистема DepthAI** — не стандартный OpenCV pipeline (хотя интеграция есть)
- **$429** — самый дорогой вариант, а в РФ ~121 000 ₽ (наценка 3x)

## Известные проблемы (Gemini 335L)

1. **Только USB 3.0** — через USB 2.0 камера может не определиться. На Jetson Orin NX все порты USB 3.2, так что проблем быть не должно
2. **OrbbecSDK v2 обязательно** — старый SDK v1 несовместим с серией Gemini 330. Это плюс (новый SDK чище), но нужно следить за версиями
3. **RGB ограничен 800p** — для SLAM/obstacle avoidance достаточно, но для задач CV с высоким разрешением (OCR, мелкие детали) может потребоваться отдельная RGB-камера
4. **Баги с C2D alignment** — некоторые stream profiles не поддерживают colour-to-depth выравнивание. Исправлено в SDK v2.4.8+

## Итоговая рекомендация

### РЕШЕНИЕ: Orbbec Gemini 335L (~$359 / ~27 000 ₽)

**Почему:**
1. **Лучшая точность** в ценовой категории (RMSE < 0.8% @ 2м)
2. **Самый широкий FOV** (90°×65°) — видит больше, меньше слепых зон
3. **IP65** — робот защищён от пыли и случайной воды
4. **< 3 Вт** — минимальная нагрузка на аккумулятор
5. **MX6800 ASIC** — depth вычисляется на камере, Jetson свободен для нейросетей
6. **Нулевой санкционный риск** — китайский производитель, стабильные поставки
7. **В 2 раза дешевле** чем RealSense в России при лучших характеристиках
8. **Активно развивающийся SDK** — OrbbecSDK v2, ROS2 Jazzy, Jetson-first стратегия

**Запасной вариант:** Orbbec Gemini 335 (~$264) — если бюджет ограничен и IP65 не критичен.
