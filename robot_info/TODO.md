## Стратегия: от настольного помощника к мобильному роботу

```
Фаза 1                    Фаза 2                    Фаза 3
СЛЫШИТ и ДУМАЕТ            + ВИДИТ                    + ДВИГАЕТСЯ
─────────────────         ─────────────────         ─────────────────
Orin NX на столе          + RPi 5 + AI HAT+         + Orin Nano
LLM + Whisper + TTS       + камера + YOLO           + лидар + depth
микрофон + динамик        + лица + жесты            + шасси + моторы
```

Каждая фаза — **самостоятельный полезный продукт**, не полуфабрикат.

## Фаза 1 — Настольный голосовой помощник

**Цель:** робот слышит, думает, отвечает голосом. Стоит на столе, без камеры и колёс.

**Железо:** только Orin NX 16GB (уже есть) + аудио.

### Софт Фазы 1

| Задача | Что делать | Процесс фреймворка |
|--------|-----------|-------------------|
| STT | faster-whisper + whisper-medium на GPU | WhisperProcess |
| LLM | llama.cpp + Qwen-2.5 14B Q4 на GPU | LLMProcess |
| TTS | Piper + ru_RU модель на CPU | TTSProcess |

### Перебалансировка при переходе к Фазе 3

При добавлении Orin Nano — **Whisper переезжает с NX на Nano:**

| Процесс | Фаза 1-2 (без Nano) | Фаза 3 (с Nano) | Почему |
|---------|---------------------|-----------------|--------|
| Whisper | Orin NX (GPU shared с LLM) | **Orin Nano** (GPU свободен) | Убрать конкуренцию с LLM |
| LLM | Orin NX | Orin NX | GPU только для LLM — быстрее |
| TTS | Orin NX | Orin NX | Лёгкий, CPU-only |
| SLAM | — | Orin Nano | Новая задача |
| Nav | — | Orin Nano | Новая задача |

## Открытые вопросы

### Механика (к Фазе 3)
- [ ] **Тип шасси:** колёсное (дифф. привод? меканум?) / гусеничное?
- [ ] **Размер платформы:** настольный (~30 см) / средний (~50 см)?
- [ ] **Манипулятор:** нужен ли? Сколько DOF? (отдельная Фаза 4?)
- [ ] **Поворот камеры:** pan-tilt на серве?

### Питание (к Фазе 3)
- [ ] **Энергобюджет:** NX (20W) + Nano (15W) + RPi (15W) + ESP32 (1W) + моторы (20-50W) = ~70-100W
- [ ] **Автономность:** цель минимум 1 час → нужно ~100 Вт·ч → LiPo 4S 7000-10000mAh
- [ ] **Распределение:** отдельные step-down для каждой платы, отдельный BEC для серв

### Софт
- [ ] **LLM модель:** Qwen-2.5 14B Q4 — проверить на Orin NX в Фазе 1
- [ ] **SLAM алгоритм:** slam_toolbox (проще) vs RTAB-Map (3D) — решить в Фазе 3
- [ ] **Симуляция:** Gazebo + ROS2 для тестирования навигации до сборки шасси
