# dataset-gen-service — универсальный сервис генерации синтетического датасета (cut-and-paste)

**Ветка:** `feat/dataset-gen-service`
**Слой:** Services (новый сервис `Services/dataset_gen`)
**Статус:** DONE 2026-06-12 (все задачи — коммит `0b91b283`)

## Цель

Универсальный движок генерации синтетического обучающего датасета методом cut-and-paste
для задач «классификация объекта + оценка угла поворота». Ядро не знает предметной области:
работает с абстракцией «класс = набор RGBA-эталонов на прозрачном фоне», вся специфика —
в YAML-пресете. В комплекте пресет под 33 русские заглавные буквы на дисках (128×128, 0–360°).

## Ключевые решения

- Один движок (`DatasetEngine.generate_sample`) на оба режима вывода: экспорт на диск и
  on-the-fly `torch.utils.data.Dataset` (torch — ленивая опциональная зависимость).
- Порядок пайплайна кадра фиксирован: эталон → поворот с expand (θ = ground truth) →
  обрезка по альфе → фон → композиция со сдвигом/масштабом → **фотометрические аугментации
  единым проходом на весь кадр после композиции** (иначе модель учит шов) → ресайз.
- Симметрия — вычисляемое свойство класса: авто-детектор (пиксельная разность поворотов
  в пределах альфы) + ручные overrides в конфиге. Кодирование угла по типу:
  `none` → (sin θ, cos θ); `180` → (sin 2θ, cos 2θ); `full` → флаг angle_valid=False.
- Windows-grабли: пути с кириллицей («А/», «Б/») → I/O только через
  `np.fromfile`+`cv2.imdecode` / `imencode`+`tofile`, не `cv2.imread`.
