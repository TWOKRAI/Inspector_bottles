Покажи текущий состав Dev Company в красивой таблице.

1. Прочитай все файлы в `.claude/agents/` (кроме `_template.md`)
2. Для каждого агента извлеки: name, model, description
3. Выведи таблицу:

```
## Dev Company — Команда

| # | Роль | Агент | Модель | Описание |
|---|------|-------|--------|----------|
| 1 | ... | ... | ... | ... |
```

4. Покажи доступные workflow-команды:
   `/plan` `/implement` `/test` `/review` `/docs` `/ship` `/pipeline` `/hire`

5. Если есть проектные команды — покажи отдельно.
