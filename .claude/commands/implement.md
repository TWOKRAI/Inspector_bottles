Запусти агента **developer** (subagent_type: "developer", model: sonnet).

Передай ему:
1. Полное ТЗ задачи (Task X.Y из плана или из $ARGUMENTS)
2. Точные пути файлов для работы
3. Acceptance criteria
4. Правила проекта из CLAUDE.md (кратко, релевантные)

Важно:
- Передавай ТОЛЬКО конкретную задачу, не весь план
- Указывай конкретные файлы и функции
- Если задача зависит от предыдущей — убедись что та выполнена

После выполнения:
- Проверь что Developer сделал коммит
- Обнови статус задачи в плане (если есть файл плана)

Задача: $ARGUMENTS
