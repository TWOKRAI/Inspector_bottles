Переиндексируй кодовую базу в qex (Qdrant + Ollama).

Вызови: `mcp__qex__index_codebase(force=True)`

**Время:** на большом репо часто **десятки минут** (эмбеддинги + Qdrant + BM25) — это нормально. Имеет смысл запускать, когда можно не держать сессию на паузе (конец дня, перерыв). Пока идёт индексация, для точного поиска по коду используй Grep.

После завершения: `mcp__qex__get_indexing_status()` и краткий результат пользователю.

Если сервисы недоступны — сначала запусти:
```bash
docker start qdrant
ollama serve
```
