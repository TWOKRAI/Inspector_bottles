---
description: Переиндексация кодовой базы в qex (с force=true для свежего индекса)
---

Перед индексацией убедись что Ollama работает:

```bash
curl -s --max-time 1 http://localhost:11434/ | grep -q running && echo UP || echo DOWN
```

Если DOWN — предложи пользователю запустить `/cold-start`, не индексируй.

Если UP — вызови tool `mcp__qex__index_codebase` с параметрами:
- `path`: абсолютный путь к корню проекта
- `force`: true (полная переиндексация)

Индексация может занять 1-5 минут на средний проект. После завершения покажи итог: число файлов, чанков, времени.
