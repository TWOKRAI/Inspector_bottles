Проверь статус qex-индекса (Qdrant + Ollama семантический поиск).

Вызови: `mcp__qex__get_indexing_status()`

Выведи:
- Статус индекса (пустой / актуальный / устаревший)
- Количество проиндексированных фрагментов
- Дату последней индексации (если есть)

Если индекс пустой или сервисы недоступны — напомни о холодном старте:
```bash
docker start qdrant
ollama serve
```
