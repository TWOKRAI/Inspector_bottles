# Рефакторинг кода

Перед рефакторингом:
1. Запусти `search_code` для поиска всех использований
2. Убедись, что есть тесты для рефакторимого кода
3. Запусти тесты — они должны проходить ДО рефакторинга

В процессе:
1. Рефакторь маленькими шагами
2. После каждого шага запускай тесты
3. Не меняй поведение, только структуру

После:
1. Запусти полный набор тестов
2. Проверь типизацию: `mypy src/`
3. Проверь линтер: `ruff check .`
