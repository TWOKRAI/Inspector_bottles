# `frontend/coordinators`

**Не** то же самое, что [`managers/`](../../managers/README.md).

| Пакет | Роль |
|-------|------|
| **`managers/`** | Домен прототипа: YAML рецептов (`RecipeManager`), права (`AccessContext`), агрегаты app-рецепта — долгоживущие сервисы и I/O. |
| **`frontend/coordinators/`** | Переиспользуемые **чистые** (или почти чистые) хелперы для нескольких панелей: парсинг слота, общие куски сценариев без дублирования в `panel_widget.py`. |

Логика конкретной фичи по-прежнему в **`presenter.py`** рядом с виджетом (MVP). Сюда выносим только то, что **дублируется** между пакетами и не является доменным менеджером.

См. также: [`docs/FRONTEND_MAP.md`](../../docs/FRONTEND_MAP.md) (раздел «Виджет, Presenter и managers»).

| Модуль | Назначение |
|--------|------------|
| [`recipe_slot.py`](recipe_slot.py) | Парсинг индекса слота рецепта. |
| [`logical_cameras.py`](logical_cameras.py) | Id логической камеры и сидирование `logical_camera_ids` / ROI / постобработки при смене типа камеры. |
