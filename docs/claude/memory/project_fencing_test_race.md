**Why:** тест утверждает то, что система выдаёт лишь по везению — это флейк по конструкции, а не регресс.

**How to apply:** детерминировать стейл нельзя инъекцией через драйвер ([[project_backend_ctl_socket_bypasses_mw]]) — штамп ставит send-мидлварь отправителя, а сокет минует receive-мидлварь.
