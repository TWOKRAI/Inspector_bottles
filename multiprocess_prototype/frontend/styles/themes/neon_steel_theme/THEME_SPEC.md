# NEON STEEL — вторая тема для PySide6 (drop-in к системе innotech_theme)

Тема-сестра Silver Industrial: тот же средне-тёмный стальной металлик и синий
акцент `#2b7fff`, но **больше объёма** (шире тональный диапазон градиентов),
**глубже тени** и **дозированный неон** (свечения как в HTML-превью v2).

Подключение: скопировать папку темы и подставить её `variables.yaml`
в существующие QSS-шаблоны (`base.qss` + `components/`). Ключи идентичны
`innotech_theme/variables.yaml` — ничего в шаблонах менять не нужно,
кроме пунктов из раздела «Ручные правки» ниже.

---

## 1. Идея темы в трёх правилах

1. **Объём через диапазон.** Каждый градиент стал контрастнее: верхняя стопа
   светлее на ~1 шаг, нижняя темнее на ~1 шаг относительно Silver Industrial.
   Пример кнопки: было `#6a7284 → #3a4150`, стало `#7c8798 → #313847`.
2. **Глубина через тени.** Все `shadow_*` подняты на +0.04…0.07 альфы,
   все `glow_*` (белые блики верхней кромки) — на +0.02…0.08.
   Контраст «светлая кромка сверху / тёмная снизу» читается сильнее.
3. **Неон дозированно.** Свечения только у: полосы под шапкой, primary-кнопок,
   активного таба, ручки скроллбара, слайдер-thumb, статусных точек.
   Обычные кнопки/панели неона НЕ получают.

## 3. Шрифты

| Роль     | Шрифт             | Загрузка |
|----------|-------------------|----------|
| Логотип  | **Square721 BdEx BT** | локальный TTF через `QFontDatabase.addApplicationFont()` |
| UI       | **Exo 2** (400/500/600/700) | Google Fonts → TTF в ресурсы |
| Моно     | **JetBrains Mono** (400/500) | как сейчас |

Exo 2 — современный техно-гротеск с полной кириллицей, близкий по духу
к Rajdhani, но менее «сплющенный». `font_family: "Exo 2"` уже в variables.yaml.

**Ручная правка** (шрифт логотипа зашит в QSS, не в переменных):
```css
QLabel#BrandLabel {
    color: @brand_color;
    font-family: "Square721 BdEx BT", "Bank Gothic(RUS BY LYAJKA)";
    font-size: @font_brand;
    font-weight: 900;
}
```

## 4. Что даёт «больше объёма» без правки шаблонов

Все градиенты в шаблонах — двух/трёхстопные с ключами из variables.yaml,
поэтому расширенный диапазон стоп работает автоматически:

- **Кнопка default:** `#7c8798 → #4e5665 → #313847` (перепад ~35% светлоты).
  Верхняя кромка `glow_2xl = rgba(255,255,255,0.24)` — заметный металлический блик.
- **GroupBox:** `#5f6a79 → #47505f → #353c49`, верхняя кромка `glow_2xl`.
- **Главное окно:** `#6a7686 → #3d4553 → #232936` — глубокое «падение» к низу.
- **Поля ввода:** колодец `#171b23 → #232937`, бордер `shadow_4xl = 0.62` —
  ощутимо темнее рамка, сильнее эффект углубления.
- **Тикер:** `#12161f → #232a38 → #1a202b` на почти чёрном бордере `0.72`.

## 5. Неон: где и сколько

| Элемент | Эффект |
|---------|--------|
| Полоса под шапкой (4px, `@accent`) | `header_underline_glow()` — blur 28, rgba(43,127,255,160) |
| Primary-кнопка | `accent_glow()` — blur 22, offset (0,10), alpha 90 → **поднять до 110** |
| Активный таб | нижний бордер 3px `@accent_hi` + `accent_glow_soft()` |
| Ручка скроллбара | градиент `#3f549b → #a7c0f2` + `accent_glow_soft()` |
| Слайдер-thumb | `accent_glow_soft()` — blur 16, alpha 75 |
| Статусные точки ok/warn/danger | `success_glow()` / `warn_glow()` / `danger_glow()` |
| BrandLabel | `accent_glow_strong()` — blur 22, alpha 140 |

Рекомендуемые правки `effects.py` для этой темы (глубже тени, ярче неон):

```python
def panel_shadow():      return _make(30, 0, 15, (0, 0, 0, 135))   # было 28/115
def deep_panel_shadow(): return _make(38, 0, 20, (0, 0, 0, 160))   # было 34/140
def button_shadow():     return _make(12, 0, 5,  (0, 0, 0, 120))   # было 10/100
def header_shadow():     return _make(20, 0, 10, (0, 0, 0, 120))
def image_slot_shadow(): return _make(32, 0, 16, (0, 0, 0, 150))
def accent_glow():       return _make(24, 0, 10, (43, 127, 255, 110))
def accent_glow_soft():  return _make(18, 0, 0,  (43, 127, 255, 90))
def accent_glow_strong():return _make(26, 0, 0,  (43, 127, 255, 160))
def header_underline_glow(): return _make(32, 0, 0, (43, 127, 255, 180))
```

## 6. Ручные правки шаблонов (2 шт., опционально)

1. **Шрифт логотипа** — см. раздел 3 (`QLabel#BrandLabel`).
2. **Неоновая кромка активного таба** — если хочется свечения как в HTML:
   в `tabs.qss` заменить `border-bottom: 3px solid @accent_hi;` — уже работает
   от переменной; свечение добавить `accent_glow_soft()` на QTabBar кодом.

Всё остальное — чистый drop-in variables.yaml.
