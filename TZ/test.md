Я разрабатываю коммерческое предложение на автоматический комплекс упаковки рулонов алюминиевой фольги.

Прилагаю файлы:
- Т_RUS_ТЗ_линия_упаковки_в_общий_гофрокороб ред.11.03.2026.docx (техническое задание)
- Опросный лист.xlsx (опросный лист с параметрами продукции)

Проанализируй требования ТЗ и предложи конфигурацию системы технического зрения и роботизированной укладки для одной технологической линии (одного потока, один перемотчик).

Исходные данные по моей части:
- Программирование буду вести на Python (multiprocessing, нейросети YOLO или свои).
- Камеры планирую использовать Hikrobot промышленные (Global Shutter, GigE Vision).
- Робот-укладчик рассматриваю либо Delta Electronics SCARA, либо дельта-робот ("паук").
- Промышленный компьютер: iROBO-3000-00i6-G5 (W680, Intel Core i7-12700 2.1 ГГц, 32 ГБ DDR4 3200 (2x16 ГБ), SSD 512 ГБ, 3xLAN (включая 2.5GbE), 2xPCIe x16, 3xPCIe x4, 2xPCI, БП 500 Вт). В этот компьютер будет дополнительно установлена дискретная видеокарта (например, NVIDIA но там особые видеокарты подходят под этот промышленный корпус) для ускорения нейросетей.

На основе ТЗ и опросного листа напиши:
1. Количество и расположение камер на одном потоке.
2. Какую модель робота лучше выбрать (SCARA или дельта) с кратким обоснованием.
3. Достаточно ли характеристик моего компьютера (с учётом добавленной видеокарты и какую посоветуешь) для обработки видеопотоков с камер в реальном времени.
4. Принципиальную схему взаимодействия: камеры -> компьютер -> робот -> укладка.

Дополнительные уточнения и рекомендации приветствуются, но основные решения должны опираться на приложенные документы.
