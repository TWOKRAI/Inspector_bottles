# 01. Обзор проекта и архитектура системы

## 1. Назначение

Автоматический комплекс упаковки бытовых рулонов алюминиевой фольги в короба и укладки коробов на паллеты. Проект для производства в г. Дмитров, Россия.

**Наша зона ответственности в КП:** система технического зрения (дефектоскопия + контроль вложений + QR/штрихкоды) + ПО управления + интеграция с ПЛК и роботами.

## 2. Масштаб

- **4 технологических линии** (4 перемотчика) в одном комплексе
- Площадь: **17.88 x 9.0 м** (~161 м²) на все 4 линии
- Режим: **24/7**, непрерывный
- 280 номенклатур продукции

## 4. Технологическая цепочка (из ТЗ)

```
Этап 1: Формирование короба          → Формирователь гофротары
Этап 2: Инспекция рулонов (ТЗ)       → Система технического зрения ◄── НАША ЗАДАЧА
Этап 3: Укладка рулонов в короб      → Робот-укладчик (SCARA/Delta)
Этап 4: Контроль вложений            → ТЗ + датчик массы           ◄── НАША ЗАДАЧА
Этап 5: Заклейка короба              → Заклейщик
Этап 6: Этикетка на короб            → Этикетировщик
Этап 7: Укладка коробов на паллет    → Робот-паллетайзер
Этап 8: Транспортировка паллета      → Конвейер
```

## 6. Зона инспекции — принцип работы (один поток)

```
  Перемотчик                  Зона инспекции                    Робот
      │                            │                              │
      ▼                            ▼                              ▼
  ═══════════╦══════════════════════════════════════╦══════════════════
  конвейер   ║   ┌───┐ ┌───┐ ┌───┐ ┌───┐          ║  зона захвата
  подачи     ║   │ R │ │ R │ │ R │ │ R │          ║  (накопитель
             ║   └───┘ └───┘ └───┘ └───┘          ║   годных)
  ═══════════╩════╤══════╤═══════════╤═════════════╩══════════════════
                  │      │           │                    │
              Cam3│  Cam1│       Cam2│                Cam4│
             торец│  верх│     верх/│                торец│
             левый│      │     бок  │                прав.│
                  │      │           │                    │
                  ▼      ▼           ▼                    │
              ┌──────────────────────┐               ┌────┘
              │  Вращающие ролики    │               │
              │  (360° осмотр)       │               │
              └──────────────────────┘               │
                                                     │
                                            ┌────────┴───┐
                                            │  Отбраковка │
                                            │  (сброс)    │
                                            └────────────┘
```

**Рулоны движутся горизонтально.** В зоне инспекции секция конвейера с вращающими роликами проворачивает рулон на 360°. Камеры сверху/сбоку снимают развёртку поверхности. Камеры с торцов снимают оба торца. Бракованные рулоны отводятся пневмосбросом или отдельным конвейером.

## 9. Требования ТЗ к ПО (что касается нас)

1. **Связь с контроллером перемотчика** — изменение скорости, остановка при сбоях
2. **Русифицированный интерфейс** с возможностью ввода новых типов продукции
3. **Архивирование данных** — срок хранения 1 год
4. **Подключение в сеть предприятия** — вывод параметров в Excel/dashboard
5. **Удалённый доступ** для диагностики
6. **Счётчики прохождения продукции**
7. **Экран аварийных сообщений** на русском с датой/временем
8. **Ручной и автоматический режимы**
9. **Система открыта** для самостоятельных корректировок заказчиком
