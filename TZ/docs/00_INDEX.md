## Ключевые решения (предварительные)

- **ПЛК:** Delta (заказчик)
- **Робот:** Delta SCARA (укладка рулонов) + промышленный 6-осевой (паллетайзер)
- **Камеры:** Hikrobot (GigE Vision, Global Shutter)
- **Вычисления:** промышленный ПК iROBO-3000 + NVIDIA GPU (основной вариант)
- **ПО:** Python, multiprocess_framework, PyQt5, YOLO/custom NN, OpenCV
- **QR/штрихкоды:** реализация через камеры ТЗ (программно), без отдельных сканеров
