# Windows — Архитектура

## Ответственность
**Окна верхнего уровня Qt**. Каждое окно — это `QMainWindow` или `QWidget`, который:
1. Собирает `Widget/`-вкладки в единый UI.
2. Получает менеджеры из `WindowManager` и раздаёт их виджетам.
3. Делегирует всю бизнес-логику менеджерам — никакой логики внутри самого окна.

---

## Файлы

| Файл | Класс | Ответственность |
|------|-------|-----------------|
| `main_window.py` | `MainWindow(QMainWindow)` | Главное окно: сборка всех вкладок (`Widget/*`), раздача менеджеров, подключение сигналов, отображение кадров |
| `loading_window.py` | `LoadingWindow(QWidget)` | Splash-экран с логотипом, прогресс-баром и процентом загрузки (новая версия) |
| `window_loading.py` | `LoadingWindow(QWidget)` | Splash-экран (старая версия, используется `WindowManager`) |
| `admin_window.py` | `PasswordDialog(QWidget)` | Диалог ввода пароля для смены уровня доступа (0 / 1 / 2) |
| `neuroun_window.py` | `NeurounWindow(QMainWindow)`, `FileManager` | Окно управления изображениями нейросети: сортировка, перемещение, удаление файлов |
| `message_window.py` | `MessageWindow(QWidget)` | Всплывающее окно с текстом, опциональным изображением и кнопками (сообщения от бота/робота) |

---

## Файлы-кандидаты на удаление / слияние

| Файл | Статус |
|------|--------|
| `main_windows_old.py` | Весь код закомментирован — исторический архив, можно удалить |
| `window_loading.py` | Старая версия `LoadingWindow`; `WindowManager` использует её — нужна до полного перехода на `loading_window.py` |
| `test_neuroun.py` | `ImageSortingWidget` — тестовый виджет, используется в `NeurounWindow` |
| `test_neuroun copy.py` | Копия тестового файла — не используется, можно удалить |
| `main_window copy.py` | Копия главного окна — не используется, можно удалить |

---

## Правила

- Окно **не содержит бизнес-логики**: если нужно что-то посчитать или сохранить — это задача менеджера.
- Окно **не создаёт** менеджеры: они передаются из `WindowManager`.
- Сигналы между окном и менеджерами — предпочтительный способ связи.
- `MainWindow` является единственной точкой сборки всех `Widget/`-вкладок.
