## Ключевые концепции

### Архитектурные слои

1. **Entry Point** - точка входа (`main.py`)
2. **App Process** - процесс приложения (`main_app.py`)
3. **Managers Layer** - менеджеры данных (`App/Managers/`)
4. **Registers Layer** - регистры управления (`App/Registers/`)
5. **UI Layer** - пользовательский интерфейс (`App/Windows/`, `App/Widget/`)
6. **Data Layer** - данные (`App/Data/`)

### Ключевые компоненты

- **DataManager** - координатор всех менеджеров данных
- **ConverterManager** - универсальная конвертация данных
- **RecipeManager** - управление рецептами (сортами)
- **CameraManager** - управление камерами
- **RegionManager** - управление регионами
- **RegistersManager** - менеджер регистров управления

### Модели данных

- **Register Models** - настройки управления (что делать)
- **Data Models** - структуры данных (как организованы данные)
