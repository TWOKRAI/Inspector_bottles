## Основные концепции

### Командный менеджер

CommandModule предоставляет систему управления командами с:
- Регистрацией обработчиков команд
- Выполнением команд через диспетчер
- Управлением метаданными и тегами
- Интеграцией с ObservableMixin

### Интеграция с BaseManager

CommandManager наследуется от BaseManager, что позволяет использовать:
- Стандартный жизненный цикл (initialize/shutdown)
- Поддержку адаптеров
- Поддержку событий
- Автоматическую статистику

### Интеграция с Dispatcher

CommandManager использует Dispatcher для диспетчеризации команд:
- Поддержка всех стратегий диспетчеризации
- Работа со сценариями
- Гибкая маршрутизация команд
