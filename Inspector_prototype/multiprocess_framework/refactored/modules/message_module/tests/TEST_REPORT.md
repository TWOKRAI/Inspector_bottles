- ✅ `Message` - основной класс

- ✅ `Priority` - enum приоритетов
- ✅ `LogLevel` - enum уровней логирования

**Разделение ответственности:**
- ✅ `Message` - только публичный API и делегирование
- ✅ `MessageValidator` - валидация (внутренний класс)
- ✅ `MessageConverter` - конвертация (внутренний класс)
- ✅ `MessageFactory` - создание и парсинг (публичный класс)
- ✅ `MessageUtils` - утилиты (внутренние функции)
