## Новые возможности

### Плагин-система

**Преимущества:**
- Легко добавлять кастомные менеджеры
- Разные менеджеры могут использовать разные плагины
- Динамическая регистрация плагинов
- Изоляция - плагины не влияют на основной код
- Переиспользование - плагины можно использовать в разных менеджерах

**Примеры использования:**
- ProcessManager с ProcessPlugin для интеграции с процессом
- RouterManager с RouterPlugin для маршрутизации сообщений
- WorkerManager с WorkerPlugin для управления воркерами
- Кастомные менеджеры с кастомными плагинами
