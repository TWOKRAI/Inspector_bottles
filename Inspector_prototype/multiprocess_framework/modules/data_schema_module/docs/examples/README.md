# Примеры использования Data Schema Module

Эта папка содержит примеры использования модуля `data_schema`, разбитые по категориям для лучшего понимания.

## Структура примеров

### 01_basic_usage.py
**Базовое использование модуля**
- Определение схем данных
- Регистрация схем в реестре
- Создание экземпляров с дефолтными значениями
- Получение дефолтных значений

**Когда использовать:** Начало работы с модулем, базовые операции.

### 02_conversion.py
**Конвертация данных между форматами**
- Конвертация моделей в dict, JSON, YAML
- Конвертация из форматов обратно в модели
- Универсальная конвертация через FormatType

**Когда использовать:** Нужна сериализация/десериализация данных, работа с файлами конфигурации.

### 03_validation.py
**Валидация данных**
- Валидация данных по схемам
- Получение ошибок валидации
- Проверка валидности без создания экземпляра
- Строгая валидация

**Когда использовать:** Нужна проверка корректности данных перед использованием.

### 04_nested_data.py
**Работа с вложенными структурами**
- Получение значений по точечной нотации
- Установка значений во вложенных структурах
- Объединение с дефолтными значениями

**Когда использовать:** Работа со сложными конфигурациями, вложенными данными.

### 05_file_operations.py
**Работа с файлами**
- Сохранение конфигураций в файлы
- Загрузка конфигураций из файлов
- Работа с различными форматами файлов

**Когда использовать:** Нужно сохранять/загружать конфигурации из файлов.

### 06_advanced.py
**Продвинутые возможности**
- Версионирование конфигураций
- Работа с ProcessData
- Использование ManagerDataAdapter
- Комплексные сценарии использования

**Когда использовать:** Продвинутые сценарии, интеграция с другими модулями системы.

## Запуск примеров

### Запуск всех примеров
```bash
python -m multiprocess_framework.modules.Shared_resources_module.data_schema.docs.examples.01_basic_usage
python -m multiprocess_framework.modules.Shared_resources_module.data_schema.docs.examples.02_conversion
# и т.д.
```

### Запуск конкретного примера
```python
from multiprocess_framework.modules.Shared_resources_module.data_schema.docs.examples import (
    example_register_schemas,
    example_create_instances
)

example_register_schemas()
example_create_instances()
```

## Рекомендуемый порядок изучения

1. **01_basic_usage.py** - начните с базовых операций
2. **02_conversion.py** - изучите конвертацию форматов
3. **03_validation.py** - разберитесь с валидацией
4. **04_nested_data.py** - работа со сложными структурами
5. **05_file_operations.py** - работа с файлами
6. **06_advanced.py** - продвинутые возможности
