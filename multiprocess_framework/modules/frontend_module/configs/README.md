# configs — конфигурационные схемы менеджеров

Pydantic-модели для инициализации ключевых менеджеров frontend-приложения.

## Ключевые символы

- `FrontendManagerConfig` — конфигурация FrontendManager (регистры, окна, профиль).
- `WindowManagerConfig` — конфигурация управления окнами (размер по умолчанию, позиция, сохранение состояния).
- `ThreadManagerConfig` — конфигурация рабочих потоков (максимум воркеров, timeout).
